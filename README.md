## Project Description

**SDG GO** is an interactive educational game designed to raise awareness about the United Nations' Sustainable Development Goals (SDGs). The game combines engaging gameplay mechanics with real-world sustainability challenges, allowing players to explore different career paths, make strategic decisions, and witness the impact of their choices on global development.

This project aims to:
- **Educate** players about the 17 Sustainable Development Goals
- **Engage** users through interactive storytelling and decision-making
- **Inspire** action toward sustainable practices and conscious choices
- **Demonstrate** the interconnectedness of global challenges and solutions

The game is built with Go, offering high performance and cross-platform compatibility.
